# Portafolio Web – Arnulfo Moreno

Portafolio profesional desarrollado con un enfoque moderno, minimalista y visualmente claro.

## 🌐 Demo
https://arnulfomo.github.io/

## ✨ Características
- Sidebar fijo con navegación clara
- Modo oscuro / claro
- Cambio de idioma (ES / EN)
- Diseño moderno y minimalista
- Estructura organizada por secciones:
  - Inicio
  - Habilidades
  - Proyectos
  - Sobre mí
  - Formación
  - Contacto

## 🛠️ Tecnologías
- HTML5
- Tailwind CSS
- JavaScript
